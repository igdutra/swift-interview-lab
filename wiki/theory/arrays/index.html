Arrays & Hashing

Theory Hub · Hash Map · Two Pointers · Sliding Window

The first and most fundamental interview block: three single-pass O(n) tools that solve the majority of array and string problems. Hash maps trade space for instant lookup; two pointers exploit sorted order to converge from both ends; sliding windows maintain an incremental summary of a moving range. Together they cover nearly every "find a pair / find a subarray" problem you will see.

Swift Code Patterns

Array Slicing

let nums = [1, 2, 3, 4, 5]

nums[..<3]      // [1, 2, 3]        — from 0 up to (not including) 3
nums[2...]      // [3, 4, 5]        — from index 2 to end
nums[1..<3]    // [2, 3]           — index 1 up to (not including) 3
nums[1...3]    // [2, 3, 4]        — index 1 through 3 inclusive
nums[...]       // [1, 2, 3, 4, 5]  — full slice

Type is ArraySlice<Int> — a zero-copy view into the original buffer. Works with any Sequence method (reduce, map, filter…).

reduce — two forms

// Form 1 — fold into a single value
nums[..<3].reduce(1, *)      // 1 * 2 * 3 = 6
nums.reduce(0, +)             // 15  — sum of all
nums.reduce(Int.min, max)     // 5   — running max; operator passed directly

// Form 2 — fold into a growing collection  (reduce(into:) mutates in place, no copies)
nums.dropLast().reduce(into: [1]) { acc, value in
    acc.append(acc.last! * value)
}
// [1, 2, 3, 4] → [1, 1, 2, 6]   (prefix products)

zip + .map(*) — combine two parallel arrays

let prefixes = [1,  1,  2,  6]
let suffixes  = [24, 12, 4,  1]

zip(prefixes, suffixes).map(*)   // [24, 12, 8, 6]
// same as .map { a, b in a * b } — * is just a (Int, Int) -> Int function

Sentinel — two ways to prepend it (prefix)

// Explicit: allocate n+1 slots, sentinel lands at index 0, loop fills index+1
var prefix = [Int](repeating: 0, count: nums.count + 1)
for index in 0 ..< nums.count {
    prefix[index + 1] = prefix[index] + nums[index]
}
// nums=[-2,0,3] → prefix=[0,-2,-2,1]  ← 0 is the sentinel at index 0

// Idiomatic: [0] IS the starting array — sentinel is already there before reduce runs
let prefix = nums.reduce(into: [0]) { acc, value in
    acc.append(acc.last! + value)
}
// same result: [0, -2, -2, 1]

Both produce length n+1 with the sentinel shoved in front. The explicit version pre-allocates and uses index+1 arithmetic. The idiomatic version starts with [0] and grows via append — same structure, no index math.

Sentinel — suffix (identity at the end)

// Explicit: sentinel at index n, loop fills backwards
var suffix = [Int](repeating: 1, count: nums.count + 1)
for index in stride(from: nums.count - 1, through: 0, by: -1) {
    suffix[index] = suffix[index + 1] * nums[index]
}
// nums=[1,2,3,4] → suffix=[24,12,4,1,1]  ← trailing 1 is the sentinel

// Idiomatic: build right-to-left, flip back — [1] seeds the right end
let suffix = nums.dropFirst().reversed().reduce(into: [1]) { acc, value in
    acc.append(acc.last! * value)
}.reversed()
// same result: [24, 12, 4, 1]

Suffix sentinel ends up at the back. The idiomatic version must reverse twice: once to feed reduce right-to-left, once to restore left-to-right order.

When to drop — the one question

// ASK: should position i be ABSENT from its own accumulated value?

// YES (product-except-self) → drop the boundary, sentinel is the identity
nums.dropLast().reduce(into: [1]) { ... }   // prefix: drop rightmost
nums.dropFirst().reversed()...              // suffix: drop leftmost
// exclusion is STRUCTURAL — i never enters its own accumulation

// NO (range sum LC303) → full array, no drop, sentinel just prepended
nums.reduce(into: [0]) { ... }              // every element is included
// exclusion happens AT QUERY TIME: prefix[right+1] - prefix[left]

If i must be absent → exclude at build time with drop. If i is included → no drop, and the query formula handles the boundary.

Full prefix/suffix pipeline (idiomatic O(n))

// nums = [1, 2, 3, 4]
let prefixes = nums.dropLast().reduce(into: [1]) { acc, value in
    acc.append(acc.last! * value)
}
// → [1, 1, 2, 6]   (sentinel [1] + products of everything LEFT of each index)

let suffixes = nums.dropFirst().reversed().reduce(into: [1]) { acc, value in
    acc.append(acc.last! * value)
}.reversed()
// → [24, 12, 4, 1]  (products of everything RIGHT of each index + sentinel [1])

zip(prefixes, suffixes).map(*)   // [24, 12, 8, 6]  ✓
