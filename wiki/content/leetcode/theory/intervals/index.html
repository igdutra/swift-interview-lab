Intervals

Theory Hub · Sort-Then-Sweep · Merge · Overlap · Coverage

Almost every interval problem collapses to one move: sort by start, then sweep once, comparing each interval only with the last committed result. The variants — merge, overlap detection, insert, gap-find, and split/handoff coverage — differ only in what the sweep records. The Overview is the single reference for the technique, its boundary gotchas (≤ vs <), and the walkthroughs that instantiate each variant.
